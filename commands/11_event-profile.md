# 目标

建立展会、活动或峰会的基础事实底稿，说明其由谁组织、如何发展、服务谁以及历届如何演变。

# 输入

- 必填：已确认的 Research Scope；活动名称；目标届次或年份。
- 选填：官方历史页面、活动手册、往届报告、场馆资料、主办方材料。
- 缺失时：按证据规则降级，不能用当前届资料替代历史事实。

# 执行步骤

1. 核实活动类型、主办方、所有权、合作机构、举办频率、地点和官方定位。
2. 整理创办背景、历史里程碑、名称或主办权变化、重大扩张或转型。
3. 按届次记录可比的观众、参展商、演讲者、国家/地区覆盖和活动规模数据。
4. 研究官方目标受众、行业覆盖、活动形式和主要功能。
5. 区分官方活动、同期活动、场外活动和品牌自办活动。
6. 标记统计定义变化、来源冲突和无法比较的数据。

# 输出

## Event Profile Brief
1. 活动身份与官方定位
2. 主办方、所有权与合作结构
3. 创办背景与历史里程碑
4. 历届地点、规模与演变
5. 官方受众与行业覆盖
6. 官方活动、同期活动与场外活动边界
7. 关键变化与 Information Gaps

每节使用：Fact / Finding / Why It Matters / Information Gap。
