# 目标

基于 Meltwater 数据识别活动往届最受媒体关注的品牌/公司、主要议题以及关注度的具体驱动因素。

# 输入

- 必填：通过门槛检查的 Meltwater 数据；已确认的关注度指标；实体名单或标准化规则；活动届次。
- 选填：参展商名单、赞助商名单、议程、发布会和奖项数据。
- 缺失时：未确认排名口径时先询问，不得自行把 Reach 或 Mentions 设为唯一标准。

# 执行步骤

1. 标准化公司、品牌、子品牌和产品名称，记录合并规则。
2. 去除或单列新闻稿重复转载，避免单一稿件造成虚假高关注。
3. 按已确认 Research Scope 中的口径计算指标，例如：独立报道数、标题提及、估算触达、互动、Share of Voice。
4. 将关注度与 Sentiment、Earned/Republished、媒体类别和市场分开呈现。
5. 识别高关注实体，并核实关注驱动：发布会、产品发布、Keynote、奖项、合作、体验、趋势议题或争议。
6. 分析主要行业议题、品牌发布主题和跨届变化。
7. 不把高报道量自动解释为正面影响或行业领导力。
