# 目标

形成目标企业的核心事实底稿，解释企业如何发展、如何经营、靠什么产品和能力创造收入。

# 输入

- 必填：已确认的 Research Scope；企业名称。
- 选填：公司资料、财报、产品材料、高管信息。
- 缺失时：按证据规则降级，并列入 Information Gap。

# 执行步骤

1. 研究成立背景、里程碑、管理层、股权与关联结构。
2. 研究业务板块、收入逻辑、客户结构、销售与交付方式。
3. 研究产品线、技术、服务、认证、专利和商业化证据。
4. 区分公司自述、第三方验证和无法核实内容。
5. 标记业务依赖、叙事矛盾和关键缺口。

# 输出

## Company Core Brief
1. 企业身份与发展历史
2. 股权、管理层与关联结构
3. 商业模式与收入逻辑
4. 产品、技术与服务能力
5. 已验证优势与能力限制
6. 关键依赖关系
7. Information Gaps

每节使用：Fact / Finding / Why It Matters / Information Gap。
