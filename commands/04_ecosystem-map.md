# 目标

绘制企业所在产业链、上下游、供应链、合作伙伴与利益相关方关系。

# 输入

- 必填：已确认的 Research Scope；Company Core Brief。
- 选填：供应商、客户、渠道、生产基地、合作项目资料。
- 缺失时：仅使用可核实公开信息并标注关系强度。

# 执行步骤

1. 划分上游、企业自身环节和下游。
2. 识别关键供应商、技术来源、生产、物流、渠道和终端。
3. 识别商业、技术、研究、政府、协会和公益合作伙伴。
4. 区分公开确认关系、历史关系、推定关系和未核实关系。
5. 标记单一依赖、控制权、声誉传导和供应链脆弱点。

# 输出

## Business Ecosystem Map
| 环节/主体 | 类别 | 与企业关系 | 关系状态 | 重要性 | 证据 | 潜在风险 | 待确认信息 |

## 关键发现
- 上游依赖
- 企业核心控制环节
- 下游触达结构
- 伙伴与利益相关方关系
- 供应链和合作风险
