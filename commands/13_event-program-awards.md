# 目标

研究活动议程、核心议题、发布会与奖项体系，识别历届公开讨论和品牌发布的主要主题。

# 输入

- 必填：已确认的 Research Scope；目标届次或比较年份。
- 选填：官方议程、会议轨道、新闻中心、品牌新闻稿、发布会日程、奖项规则和获奖名单。
- 缺失时：未找到发布证据时不得推定品牌在活动期间举行过发布会。

# 执行步骤

1. 整理年度主题、官方议程、会议轨道、Keynote 和重要论坛。
2. 聚类主要行业议题，并比较不同届次的新增、持续和减弱主题。
3. 核实产品发布、公司公告、发布会和品牌自办活动的时间、形式与主题。
4. 区分活动官方奖项、官方合作奖项、媒体/伙伴奖项和独立第三方奖项。
5. 对奖项记录主办方、资格、评审机制、类别、申报时间和获奖者。
6. 区分正式发布、预热、现场展示、概念演示和活动后公告。

# 输出

## Event Program & Awards Brief
1. 年度主题与议程架构
2. 主要议题及跨届变化
3. Keynote、论坛和重点活动
4. 发布会、产品发布与公告主题
5. Awards Map
