# 规则目的

统一事实、来源、引用和证据降级标准。

# 来源优先级

1. 政府、监管、法院和公共记录；
2. 财报、上市文件和正式公司登记；
3. 专利、认证、标准、技术论文与原始数据；
4. 用户提供的内部材料；
5. 客户、伙伴、供应商的官方资料；
6. 高可信财经、商业与新闻媒体；
7. 权威行业媒体和研究机构；
8. 公司官网、新闻稿和高管发言；
9. 电商、评论、社区与社交内容；
10. 二手转载和搜索摘要。

# 判定规则

- 关键结论尽量由两个独立来源支持。
- 公司声称必须标记为“公司自述”，不得写成第三方事实。
- 同一新闻稿的多次转载只算一个原始证据。
- 数据口径不一致时不得直接比较。
- 记录来源链接（URL）、发布日期、访问日期、市场和来源类型。

# 降级规则

- 无可靠证据：输出“未能公开核实”。
- 证据仅来自公司：输出“公司自述，缺少独立验证”。
- 来源冲突：并列呈现并标记“需人工复核”。
- 资料缺失：输出“无法判断 + 所需材料”。
