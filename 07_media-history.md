# 目标

分析企业、竞品和行业过去如何被媒体与公开内容呈现，识别稳定叙事、缺失叙事和争议主题。

# 输入

- 必填：Research Charter；分析对象和时间范围。
- 选填：历史报道清单、新闻稿、媒体数据库导出。
- 缺失时：公开样本不足则明确样本限制。

# 执行步骤

1. 收集并区分独立报道、采访、测评、导购、新闻稿转载和付费内容。
2. 按时间、市场、媒体类别、主题、业务线、发言人和情绪分类。
3. 分析公司自述与媒体叙事是否一致。
4. 比较不同市场、不同阶段或竞品的历史叙事。
5. 仅给出媒体类别及代表性示例，不生成执行名单。

# 输出

## Historical Media Dataset
| Date | Company | Market | Media | Media Category | Link | Topic | Business/Product | Format | Sentiment | Key Message | Earned/Paid/Republished | Notes |

## Narrative Findings
- 高频主题
- 低认知或缺失主题
- 叙事差异
- 争议或负面主题
- 后续可进一步研究的媒体类别与代表性示例
